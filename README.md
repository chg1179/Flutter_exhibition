## 🔎프로젝트 소개

# 커뮤니티를 통한 유저들 간의 소통, 전시회 정보 공유를 할 수 있는 별도 후기 게시판 서비스, 캘린더를 통한 팔로잉 친구와 기록 공유 서비스 제공
<br>

# 코로나 종식 이후 전시회, 축제 등과 같은 문화 생활의 수요 증가에 발맞춰 전시 정보를 모아볼 수 있는 맞춤형 서비스 제공
<br>

# 어플로도 전시를 체험할 수 있는 느낌으로 작품 이미지와 정보를 제공
